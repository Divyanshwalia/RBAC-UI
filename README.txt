Role-Based Access Control (RBAC) UI - Admin Dashboard
This project is an implementation of a Role-Based Access Control (RBAC) system with a user-friendly Admin Dashboard. It allows administrators to manage users, roles, and permissions with ease. The interface is built with React JS and styled using Tailwind CSS to provide a modern and responsive design.

Features:

User Management

View a list of users with options to add, edit, or delete users.
Assign roles to users.
Manage user status (Active/Inactive).

Role Management

Define and edit roles.
Assign permissions (Read, Write, Delete) to each role.
Permissions Management

Easily assign or modify permissions for roles.
Permissions are displayed clearly for quick understanding and modification.


Mock API calls to simulate CRUD operations for users and roles.


The user interface is designed with Tailwind CSS, ensuring a responsive layout and modern aesthetics.

Tech Stack
Frontend: React JS, Tailwind CSS
State Management: React useState (for handling state locally)
API Simulation: Mock API calls using fetch 


User Management:

View all users in the system.
Add new users by clicking the Add User button.
Edit existing users or delete them.
Assign roles to users (e.g., Admin, User).

Role Management:

Define new roles like Admin, Manager, or User.
Set permissions for each role (Read, Write, Delete).
Modify or delete roles.

Permissions Management:

For each role, define the actions allowed (e.g., Read and Write permissions for a role).
API Simulation (Optional):

Mock API calls simulate interactions with the backend for CRUD operations.
Design
The design is built using Tailwind CSS, ensuring a clean, minimal, and modern look.
The layout is responsive, adapting to different screen sizes (desktop, tablet, and mobile).
